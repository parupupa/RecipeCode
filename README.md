# RecipeCode
### https://recipecode.onrender.com

## 1. サービス概要
このサービスは、レシピを保存するだけでなく、料理をするたびに加えた工夫や調整内容を記録し、改良履歴として蓄積できるアプリです。
ユーザーは変更した部分のみを簡単に入力するだけで、その時点の完成形レシピが自動的に保存され、過去の試行錯誤を振り返ることができます。
これにより、毎回同じ調整を繰り返す手間や、記録の分散による管理の煩雑さを解消し、自分に合ったレシピを効率よく再現・改善できるようになります。
料理中でも使いやすいシンプルな操作性を重視し、継続的に記録しやすい設計とすることで、日常の調理をより快適で前向きな体験に変えることを目指します。

---

## 2. このアイデアはどこから生まれたか
### きっかけとなった体験・感情
私はお菓子作りが好きで、同じ名前のお菓子でもいろいろなレシピを試したり、少しずつ材料や入れる量を変えて、自分の理想のレシピを作ることがあります。
いつもはスマホのメモ帳で毎回元のレシピをコピーして、その内容を書き換えて保存をしています。
何度もコピーが必要なことや、体裁が見づらくなることからエクセルなどで整理することも考えましたが、調理時にエクセルを使用するのは大変だと感じ、結局メモ帳を使っていました。
レシピの改良履歴をもっと簡単に見やすく残すことができないのだろうかと考えました。
